API 참조 문서

API 참조 문서는 Node.js의 함수 또는 객체에 대한 자세한 정보를 제공합니다. 이 문서는 메서드가 받아들이는 인수, 메서드의 반환 값과 해당 메서드와 관련된 오류를 나타냅니다. 또한 Node.js의 다른 버전에서도 사용할 수 있는 방법을 나타냅니다.

이 문서는 Node.js가 제공하는 내장 모듈에 대해서 설명합니다. 커뮤니티에서 제공하는 모듈은 문서화하지 않습니다.
